# react-native-version-checker

Reliable app version checking and update prompts for React Native (iOS & Android).

## Installation


```sh
npm install react-native-version-checker
```


## Usage


```js
import { multiply } from 'react-native-version-checker';

// ...

const result = multiply(3, 7);
```
